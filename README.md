# fract-ol

Fract-ol is an interactive graphic program that generate fractal images using MiniLibX library. It allows users to visualize and explore the Mandelbrot set, Julia sets, and Tricorn fractal in real-time.

## Goal
Create fractal images using the MiniLibX library, enable zooming in and out, implement multiple color palettes, ensure a smooth window interface, and allow the program to exit cleanly without memory leaks.

## Usage
1. Clone the repository.
2. Command "make" to compile the program (mandatory and bonus).
3. Run executable "./fractol" to display options menu.
